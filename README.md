# 2026 Clock

A simple web-based clock application built with HTML, CSS, and JavaScript.

## Features

- Displays current time with hour, minute, and second hands
- Clock ticks for hour and minute markers
- Equation generator for mathematical elements ( educational purposes)
- Responsive design

## Project Structure

- `index.html`: Main HTML file for the clock interface
- `clock.css`: Stylesheet for clock appearance
- `js/clock_hand_update.js`: JavaScript for updating clock hands
- `js/clock_tick_placement.js`: JavaScript for placing clock ticks
- `js/eq_generator.js`: JavaScript for generating equations
